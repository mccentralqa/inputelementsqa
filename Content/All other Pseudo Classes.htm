:first-child

 

This is the first child of the div- Text color should be red

This is the last child of the div- should have no color

 

:last-child

 

This is the first child of the div- should have no color

This is the last child of the div- text color should be green

 

:nth-child(n)

In this example, we have pseudo class :nth-child set so that every 3rd element in the list will have a lime green color.

:nth-child(3n)

  • first
  • second
  • third- Text color should be lime green
  • fourth
  • fifth
  • sixth- Text color should be lime green

 

:nth-last-child(n)

In this example, we have pseudo class :nth-last-child set so that 5th element in the list will be highlighted

:nth-last-child(2)

  1. first
  2. second
  3. third
  4. fourth
  5. fifth- this element should be highlighted in yellow
  6. sixth

 

:only-child

Text should be red as this is the only child of this div

:nth-last-of-type(n)

:not(selector)

In this example, the first “I should...” line of text is set to the "blue" class and should render blue. The next line is just a regular paragraph, but because we used :not(#blue), any p in this div that does not use the "blue" class will render red and 20pt

I should be blue text because I belong to the "blue" class

I should be 20pt and red text because I do not belong to the "blue" class
